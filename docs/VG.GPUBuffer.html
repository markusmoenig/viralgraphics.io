Class: GPUBuffer

VG. GPUBuffer

new GPUBuffer(type, size [, dynamic] [, isIndexBuffer])

GPUBuffer encapsulates OpenGL VBO
Usage:

var g = new GPUBuffer(VG.Type.Float, 3*3).create();
... fill with data ...
g.update(); // transfer data and optional alloc
...
... link data to shader pointer ...
g.vertexAttrib(...);
...
g.bind();
g.update();
... drawing ...
shader.bind();
b.drawBuffer(...);
...
g.destroy()
g.create()
...
g.dispose()
