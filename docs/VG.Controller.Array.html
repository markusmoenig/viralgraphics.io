Class: Array

VG.Controller.Array

new Array()

Creates an array controller which handles objects inside a flat array.
You will not need to create this controller yourself, it is mostly returned by widgets like VG.UI.ListWidget.
Supports the following observer keys: "parentSelectionChanged", "selectionChanged", "changed", "addItem", "removeItem".
Properties:
Name Type Description
selected object The currently selected object.
selection array The currently selected objects when multiSelection is on.
multiSelection bool Set to true if this controller should handle multiple selections, default is false.
length number The number of items managed by the controller (read-only).
