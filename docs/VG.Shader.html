Class: Shader

VG. Shader

new Shader(vSource, fSource)

Construct & compile shader with given source.

    Example:
    s = new Shader(vsSource, fsSource);
    s.create();
    ...
    // drawing
    s.bind();
    gpuBuffer.drawBuffer(...);
    ...
    // clean up
    s.destroy();
Parameters:
Name Type Description
vSource string

Vertex source
fSource string

Fragment source

<static, readonly> Blend :number

Shader Blend mode.

Type:
  • number
Properties:
Name Type Default Description
None number 0

No blending.
Alpha number 1

Multiply alpha on both source and destination.
Brush number 2

Use alpha on source, one for destination.
OneAlpha number 3

Suitable for transparency.
ReverseBrush number 4

Use alpha on source, zero for destination.
Additive number 5

Simply add source and destination.
AdditiveIgnoreAlpha number 6

Add source and destination but alpha.
RemoveBack number 7

Ignore back. Back will become dark at places with no source.
