Methods

storeDataForPath(path, value, noUndo, forceStorage, undoText) → {VG.Data.UndoItem}

Stores a given value for the given path and creates an undo step. Widgets like a Checkbox widget use this function to store user changes inside the model. However this function can also be used directly to store value changes.
Parameters:
Name Type Description
path string The path of the value to store.
value object The data value to store inside the model.
noUndo bool Optional, if true blocks the creation of an undo step for the value change.
forceStorage bool Optional, if true forces storing of the value. Normally if the model value is the same as the new value
undoText undoText An optional custom text for this undo step. this function will not store it, however to make sure this value is always stored set forceStorage to true. This is useful for undo groups where the initial value may be the same as in the model but other value changes inside the group will change values.

Returns:
The undo item created for this data value step. To add further undo steps into this group just call addSubItem() on the undo item.
